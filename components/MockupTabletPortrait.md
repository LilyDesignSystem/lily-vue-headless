# MockupTabletPortrait

A box area that looks like a tablet computer in portrait mode. Renders a decorative `<div>` container the consumer styles to look like the named device.

## Props

- default slot — the content rendered inside the frame

## Usage

```vue
<MockupTabletPortrait>
  <img src="screenshot.png" alt="App screen" />
</MockupTabletPortrait>
```
