# MockupBrowser

A box area that looks like a web browser. Renders a decorative `<div>` container the consumer styles to look like the named device.

## Props

- default slot — the content rendered inside the frame

## Usage

```vue
<MockupBrowser>
  <img src="screenshot.png" alt="App screen" />
</MockupBrowser>
```
