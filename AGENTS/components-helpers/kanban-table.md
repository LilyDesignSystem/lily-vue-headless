# KanbanTable

Pattern:

```tsx
<KanbanTable>
    <KanbanTableHead>
        <KanbanTableTR>
            <KanbanTableTH>…</KanbanTableTH>
        </KanbanTableTR>
    </KanbanTableHead>
    <KanbanTableBody>
        <KanbanTableTR>
            <KanbanTableTD>…</KanbanTableTD>
        </KanbanTableTR>
    </KanbanTableBody>
    <KanbanTableTfoot>
        <KanbanTableTR>
            <KanbanTableTH>…</KanbanTableTH>
        </KanbanTableTR>
    </KanbanTableTfoot>
</KanbanTable>
```
